Welcome to Burning OKR!

Burning OKR unterstützt Sie bei der Erreichung strategischer Unternehmensziele und fördert eine agile und transparente Unternehmenskultur. Das OpenSource OKR Tool basiert dabei auf den langjährigen Erfahrungen der BROCKHAUS AG mit dem OKR Framework.

Visionen mit Hilfe von OKR verwirklichen

Das agile OKR Framework unterstützt Sie durch einen strukturierten Zielvereinbarungsprozess bei der Erreichung der strategischen Unternehmensziele. Kurze Zyklen und messbare Ziele fördern die regelmäßige Evaluierung der Ergebnisse und bieten Ihnen die Möglichkeit, Ihr Geschäftsmodell anhand von neuen Informationen schnell zu modifizieren. Mit Burning OKR bieten wir Ihnen ein leistungsstarkes und nutzerfreundliches OpenSource Tool zur Unterstützung.

Digitalisieren Sie Ihren gesamten OKR-Prozess

Mit Burning OKR bilden Sie alle essenziellen Bestandteile des OKR Frameworks digital ab und schaffen teamübergreifende Transparenz. Die Teams verwalten ihre eigenen Objectives und Key Results mit Burning OKR und machen ihre Fortschritte und Erfolge sichtbar. Darüber hinaus können innerhalb des Tools die Hierarchien Ihres Unternehmens sichtbar gemacht werden.

Verwalten und steuern Sie Ihre Unternehmensziele

Objectives und Key Results werden in Zyklen bearbeitet. Gestalten Sie Name, Länge und Inhalte der Zyklen nach Ihren Wünschen. Behalten Sie den Überblick über die Produktivität der OKR Teams durch die Fortschrittsanzeige der Objectives und Key Results.

Innovative Features

Unser OKR Tool ermöglicht es Ihnen, mitsamt aller notwendigen Features, den OKR Prozess und dessen Ergebniskennzahlen ganzheitlich zu digitalisieren. Im Detail bietet Burning OKR…

  • die Authentifizierung über Azure Active Directory oder einer lokale Nutzerverwaltung
  • die Verwaltung von mehreren Firmen (Multitenancy)
  • das Abbilden von Zyklen, Firmenstrukturen, Teams, Objectives und Key Results
  • eine Fortschrittsanzeige für Objectives und Key Results
  • eine flexible Individualisierung, u.a. die Bezeichnung von Teams, die Einschränkung der maximalen Key Result Anzahl etc.
  • einen Datenexport als Excel-Tabelle

Einfache Installation

Burning OKR lässt sich einfach auf verschiedenen Plattformen installieren. Wir bieten Ihnen Tutorials zur Installation mit...

  • Docker
  • einem Tomcat Web Server (coming soon)

Unser kostenloses Angebot für Sie

Die Anwendung ist für Firmen- und Privatkunden komplett kostenlos. Außerdem können Sie den Open Source Code jederzeit auf GitHub einsehen und an Ihre Bedürfnisse anpassen. Gerne stehen wir Ihnen dabei unterstützend zur Seite. Machen Sie Ihre Anpassungen als Pull Request für die ganze Welt verfügbar!

Wie Sie uns erreichen

Sie haben Fragen, Anregungen, oder Probleme mit dem Burning OKR Tool? Gerne können Sie uns kontaktieren.

Bei Bugs oder Verbesserungsvorschlägen hinsichtlich des Tools kontaktieren Sie uns bitte über GitHub. Wenn die BROCKHAUS AG für Sie eine Individualisierung vornehmen soll, Sie eine Beratung oder eine andere Dienstleistung benötigen, kontaktieren Sie uns bitte über info@brockhaus-ag.de.
